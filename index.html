What is this?

This page is created and maintained entirely by autonomous Claude cloud agents — no human pushes code here. Two agents run on a schedule: one creates content updates and opens pull requests, the other reviews them, merges approved changes, and verifies the live site.

The source is at github.com/bsord/hello-agent-pages. Follow the steps below to build the same thing yourself.

How the agent loop works

Creator Runs every 2 hours

Clones the repo, checks for open PRs with feedback, implements any requested changes, then pushes a new branch and opens a PR with updated content.

Reviewer Runs every 2 hours (30 min offset)

Finds open PRs, reviews the diff, posts feedback if changes are needed, or approves and merges if the PR looks good. After merging, fetches the live Pages URL to verify the update went through.

Set this up yourself

1

Install Claude Code

Install the Claude Code CLI and open it in any directory.

npm install -g @anthropic-ai/claude-code
claude
2

Install and authenticate the GitHub CLI

Claude Code needs gh to create repos, open PRs, and push commits on your behalf.

# Install gh (Ubuntu/Debian)
sudo apt install gh

# Authenticate (opens browser)
! gh auth login

In Claude Code, prefix commands with ! to run them interactively in your terminal.

3

Create a public GitHub repo

Ask Claude Code to create the repo. It will use your gh auth automatically.

gh repo create hello-agent-pages --public --add-readme
4

Enable GitHub Pages

Turn on Pages via the API — deploy from the main branch root.

gh api repos/YOUR_USERNAME/hello-agent-pages/pages \
  --method POST \
  -f source.branch=main \
  -f source.path=/
5

Push your initial index.html

Clone the repo, add your page, commit, and push to main.

git clone https://github.com/YOUR_USERNAME/hello-agent-pages.git
cd hello-agent-pages
# create index.html
git add index.html && git commit -m "initial page"
git push
6

Get your GitHub token for the cloud agents

The cloud agents need a token to push commits. Grab your current gh token:

gh auth token

Copy it — you'll embed it in the agent prompts in the next step.

7

Create the Creator cloud agent

In Claude Code, use /schedule to create a routine that runs every 2 hours. The prompt should tell the agent to: clone the repo using the token, update content, push a branch, and open a PR.

/schedule
8

Create the Reviewer cloud agent

Create a second routine offset by 30 minutes (30 */2 * * *). This agent finds open PRs, reviews them, merges if approved, and verifies the live URL.

/schedule
9

Watch it run

Visit claude.ai/code/routines to see your agents, check run history, and trigger manual runs. The status bar at the top of this page updates every time the agents push a merge.
